Virtual Control & distributed systems

Basic Javascript with HTML & CSS

start learning javascript with basic concepts like the console, statements, primitive types, scope, operators, rest and spread syntax, loops, array methods map() push() functions, arrow functions, high-order functions, HTML elements, prototype, objects oriented programing(OOP), events, network request application dealing with backend, event loop & call stack.

cover HTML especially the type of html, head, body, header tags(h1 to h6), p tags, navigation link, footer, radio button, checkbox, id , classes among others.

learn basic CSS styling,applied visual design, applied accessibility & css flexbox for web design.
